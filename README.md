# Website for the R Package GAMLC

Thank you for your interest in the GAMLC R package.  

This package is designed for **Graphical Models for Mixed-Type Data: Latent Gaussianization with Covariates**. The website for the package is available in this repository as a downloadable ZIP file for your convenience.  

We welcome any questions, feedback, or discussions regarding the package.
